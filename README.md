# Olist Customer Analysis Dashboard

Dashboard analisis distribusi customer Olist E-Commerce berdasarkan wilayah di Brasil.

## Struktur Proyek

```
submission/
├── data/
│   └── olist_customers_dataset.csv
├── dashboard/
│   └── dashboard.py
├── notebook.ipynb
├── requirements.txt
├── README.md
└── url.txt
```

## Cara Menjalankan Dashboard

### 1. Clone Repository
```bash
git clone https://github.com/username/nama-repo.git
cd nama-repo
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Jalankan Dashboard
```bash
streamlit run dashboard/dashboard.py
```

### 4. Buka Browser
Dashboard akan otomatis terbuka di:
```
http://localhost:8501
```
